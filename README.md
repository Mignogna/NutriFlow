# **🚀 NutriFlow - Landing Page de Comida Saudável**
Uma landing page moderna, responsiva e interativa desenvolvida para entusiastas de gastronomia e vida saudável. O projeto apresenta um design elegante com animações de rotação (3D flip) para exibir detalhes dos pratos.

## **🎯 Funcionalidades**
**Design Interativo:** Banner central com efeito de rotação para exibir descrição e preço dos pratos.

**Troca de Imagens:** Seleção dinâmica de pratos através de miniaturas laterais.

**Busca Expansível**: Barra de pesquisa animada no cabeçalho.

**Totalmente Responsivo:** Adaptado para dispositivos móveis, tablets e desktop.

**Ícones Dinâmicos:** Integração com IonIcons para uma interface moderna.

## **🛠️ Tecnologias Utilizadas**
**HTML5:** Estruturação semântica do conteúdo.

**CSS3:** Estilização avançada com Variáveis (Custom Properties), Flexbox e Animações @keyframes.

**JavaScript:** Manipulação do DOM para troca de conteúdo e estados de botões.

**IonIcons:** Biblioteca de ícones vetoriais.

**Google Fonts:** Tipografia customizada (Roboto e Berkshire Swash).

## **📸 Demonstração do Layout**
O projeto utiliza um sistema de **cartão 3D** onde:

A frente mostra a imagem do prato em destaque.

Ao clicar em "Learn More", o cartão gira 180° revelando a descrição, preço e botões de download.<hr>

# **🚀 NutriFlow - Healthy Food Landing Page**
A modern, responsive, and interactive landing page built for food enthusiasts and healthy lifestyle lovers. This project features an elegant design with 3D flip animations to showcase dish details.

## **🎯 Features**
**Interactive Design:** Central banner with a 3D rotation effect to display descriptions and pricing.

**Dynamic Image Selection:** Switch between different dishes seamlessly using side thumbnails.

**Expandable Search:** Animated search bar in the header for better UX.

**Fully Responsive:** Optimized for mobile, tablets, and desktop screens.

**Modern Icons:** Integrated with IonIcons for a clean and professional interface.

## **🛠️ Technologies Used**
**HTML5:** Semantic structure for better SEO and accessibility.

**CSS3:** Advanced styling using CSS Variables, Flexbox, and @keyframes animations.

**JavaScript:** DOM manipulation for content switching and button state management.

**IonIcons:** High-quality vector icons.

**Google Fonts:** Custom typography (Roboto and Berkshire Swash).
